HTML Converter

Plugin Identifier: 20000

Summary

This is a cross platform plugin to convert HTML to PDF, JPG, PNG, BMP or SVG using the wkhtmltopdf library.

Convert

HTML Convert to image

image:=HTML Convert to image (html; keys; values)

html (TEXT)

The HTML content, path to html, or url.

keys (ARRAY)

Option keys.

values (ARRAY)

Option values.

image (BLOB)

Image data.

Discussion

HTML Convert to pdf

pdf:=HTML Convert to pdf (html; keys; values)

html (TEXT)

The HTML content, path to html, or url.

keys (ARRAY)

Option keys.

values (ARRAY)

Option values.

pdf (BLOB)

PDF data.
